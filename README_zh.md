# Project QuickOpen for Command Palette

[English](README.md) | [中文](README_zh.md)

### 概述

从 CmdPal 的命令面板中立即搜索并打开您最近的 JetBrains 项目。

![Example.png](images/Example.png)

### 功能特性

- **🚀 快速项目访问**: 即时搜索并打开所有 JetBrains IDE 的最近项目
